Support & Contact

If you think you found an error or you have an improvment for PHP Depend, you can send questions and suggestions to our mailinglist. Or if you are interested in helping to improve this software, by writing patches, extensions or just providing new ideas, you should feel free to join PHP Depend's IRC channel or send a message to the users@pdepend.org mailinglist.

IRC

  • You will always find help on PHP Depend related topics in the #pdepend channel on the FreeNode network.

Issue-Tracker

  • You can file a ticket in PHP Depend's Issue tracker when you think you found a bug or you have a feature request for PHP Depend.
